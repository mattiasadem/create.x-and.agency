## Core Principles

### 1) HOOK FIRST
- First 3 seconds decide everything
- Lead with the payoff, not the setup
- Pattern interrupt to stop the scroll

### 2) ONE IDEA PER PIECE
- Single focused message
- Clear takeaway
- Memorable, shareable

### 3) PLATFORM NATIVE
- Different content for different platforms
- Respect each platform's culture
- Adapt format, not just dimensions

## Hook Formulas

### Curiosity
```
"Nobody talks about [X], but..."
"I discovered [X] and it changed everything."
"The real reason [common belief] is wrong."
```

### Contrarian
```
"Stop [common advice]."
"[Popular thing] is overrated. Here's why."
"Unpopular opinion: [take]"
```

### Results
```
"I [achieved X] in [timeframe]. Here's how."
"This [method] got me [result]."
"[Number] [thing] that [benefit]."
```

### Questions
```
"What if [surprising possibility]?"
"Why does everyone [behavior] when [alternative] works better?"
```
